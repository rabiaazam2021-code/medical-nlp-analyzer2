Medical NLP Sentiment Analyzer 🩺
This project analyzes medical reviews to predict patient sentiment (Positive/Negative) using Machine Learning.

✨ Key Features:
Backend: Trained a Multinomial Naive Bayes model on medical review data.

Frontend: Built an interactive dashboard using Streamlit.

Integration: Real-time sentiment prediction from text input.

🛠️ How to Run:
pip install -r requirements.txt

streamlit run app.py
